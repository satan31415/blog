### 前端技术路线图
```
- HTML
    - HTML5
- CSS
    - 预编译器
        - SASS
        - LESS
    - 框架
        - Bootstrap
        - Tailwind CSS
        - Uno CSS
- JavaScript
    - TypeScript
    - jQuery
    - MVVM 框架
        - React
            - Umi
            - Dva
            - Redux
            - axios
            - ahooks
            - UI
                - Ant Design
            - SSR
                - Next.js
        - Vue
            - Vuex
            - Vue Router
            - Pinia
            - axios
            - UI
                - Element、Element Plus
                - Ant Design Vue
                - uview、uview-plus
            - SSR
                - Nuxt.js
- 工程化
    - Webpack
    - Vite
- 浏览器
- 网络
    - HTTP
    - HTTPS
    - WebSocket
    - Fetch API
- 代码版本控制
    - Git
- 多端开发
    - uniapp(小程序, H5)
    - flutter(安卓, ios)
    - electron(桌面端)
- 服务端
    - Express
    - Nest.js
```


### 前端一些名词
- SPA: Single Page Application, 单页面应用
- SSR: Server Side Render, 服务端渲染
- MVVM: Model-View-ViewModel, 模型-视图-视图模型
